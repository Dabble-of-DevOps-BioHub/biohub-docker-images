Welcome to Dabble of DevOps Bioinformatics Jupyterhub Docker Images's documentation!
====================================================================================

WIP

This project is meant to supply preconfigured Docker images for various Bioinformatics analysis types and needs.

Each image comes with `RStudio Server <https://rstudio.com/products/rstudio/download-server/>`_ and the `Jupyter RSession Proxy <https://github.com/jupyterhub/jupyter-rsession-proxy>`_ package installed.
